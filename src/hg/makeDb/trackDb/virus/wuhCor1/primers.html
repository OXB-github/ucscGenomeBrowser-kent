Description

This track shows the locations of those primers in detection kits that match the reference sequence. The primers were copied from the file all-COVID-19-kit-primers.fasta.

All are RT-PCR primer sets. Each set consists of one forward, one reverse and one internal primer.

Four primers match the reference sequence over less than 14bp: 2019-nCoV_N2-P, HKU-ORF-1b-nsp14-R, 2019-nCoV_N1-R, WH-NIC_N-P. Four primers were not found at all: HKU-ORF-1b-nsp14-F, 2019-nCoV_RP-P, 2019-nCoV_RP-R, 2019-nCoV_RP-F.

Here is our current understanding of the origin of the primers:

Prefix Institution Overview Manual
China- and Orf1b China CDC
NIID- Nat. Inst. of Infect. Dis., Japan
WH- National Inst. of Health (Thailand)
HKU- The University of Hong King Detects N gene and Orf1b. Not specific for SARS-Cov2, but other Sarbecovirus species are not in circulation WHO Peiris Protocol
2019-nCoV- CDC, USA Three reactions, target: N gene. One primer/probe set detects all betacoronaviruses, two sets are specific for SARS-CoV-2. All 3 assays must be positive Instructions
EU-Drosten Drosten Lab, Charite Berlin, Germany Set 1: run E and RdRp primers (designed for SARS-CoV, SARS-CoV-2, and bat-associated betacoronaviruses), if Set 1 is positive, use SARS-Cov-2 specific detection primer Instructions

Summary

Bioinformatic formats for currently-deployed primer-probe sets for detecting COVID-19 (currently available in FASTA format). The sequences and the identifiers for these primers were obtained from the following sources:

  • CDC Laboratory Guidance
  • Comparative analysis of primer-probe sets for the laboratory confirmation of SARS-CoV-2
  • Detection of 2019 novel coronavirus (2019-nCoV) by real-time RT-PCR

Methods

The primers were mapped with the following command:
"blat ../../wuhCor1.2bit primers.fa stdout -stepSize=1 -tileSize=6 -minScore=10 -oneOff=1 -noHead > primers.psl"
