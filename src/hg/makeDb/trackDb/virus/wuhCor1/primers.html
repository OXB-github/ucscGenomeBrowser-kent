Description

This track shows the locations of those primers in detection kits that match the reference sequence. The primers were copied from the file all-COVID-19-kit-primers.fasta".

Four primers match the reference sequence over less than 14bp: 2019-nCoV_N2-P, HKU-ORF-1b-nsp14-R, 2019-nCoV_N1-R, WH-NIC_N-P. Four primers were not found at all: HKU-ORF-1b-nsp14-F, 2019-nCoV_RP-P, 2019-nCoV_RP-R, 2019-nCoV_RP-F.

Summary

Bioinformatic formats for currently-deployed primer-probe sets for detecting COVID-19 (currently available in FASTA format). The sequences and the identifiers for these primers were obtained from the following sources:

  • CDC Laboratory Guidance
  • Comparative analysis of primer-probe sets for the laboratory confirmation of SARS-CoV-2
  • Detection of 2019 novel coronavirus (2019-nCoV) by real-time RT-PCR

Methods

The primers were mapped with the following command: "blat ../../wuhCor1.2bit primers.fa stdout -stepSize=1 -tileSize=6 -minScore=10 -oneOff=1 -noHead > primers.psl"
