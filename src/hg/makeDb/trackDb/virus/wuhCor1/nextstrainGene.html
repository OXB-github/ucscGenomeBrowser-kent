Description

Nextstrain.org displays data about protein-coding variants that occur in the current 2019/2020 outbreak. Since next-strain shows only protein coding variants but this Genome Browser shows the whole genome, we have added a genome track to show which part of the genome is covered by Nextstrain.

Use this track in conjunction with the tracks Nextstrain Vars (all variants) and Nextstrain Clades (the main classes of variants as defined by Nextstrain).

Methods

Data was downloaded from Nextstrain, file ncov.json.
