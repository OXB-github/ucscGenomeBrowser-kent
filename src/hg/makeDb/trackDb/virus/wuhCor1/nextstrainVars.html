Description

Nextstrain.org displays data about protein-coding variants that occur in the current 2019/2020 outbreak. This track shows all the Nextstrain sequence variants. It is updated daily. The original source of the full viral sequences are NCBI Genbank and GISAID.

Use this track in conjunction with the tracks Nextstrain Genes (parts of genome covered by Nextstrain) and Nextstrain Clades (the main classes of variants as defined by Nextstrain).

Summary

Methods

Data was downloaded from Nextstrain.org, from the file ncov.json and converted to VCF.

Data Access

You can download the VCF file underlying this track from our Download Server.

The raw data can be explored interactively with the Table Browser or the Data Integrator. The data can be accessed from scripts through our API, the track name is "nextstrainVars".

Data usage policy

The data presented here is intended to rapidly disseminate analysis of important pathogens. Unpublished data is included with permission of the data generators, and does not impact their right to publish. Please contact the respective authors (available via the Nextstrain TSV file) if you intend to carry out further research using their data. Derived data, such as phylogenies, can be downloaded below - please contact the relevant authors where appropriate.
