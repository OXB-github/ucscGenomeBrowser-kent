Description

Nextstrain.org displays data about protein-coding variants that occur in the current 2019/2020 outbreak. To simplify the display and show that the different sequences fall into different groups, Nextstrain has created "clades" of sequences. This track shows these clades.

Use this track in conjunction with the tracks Nextstrain Genes (part of genome covered by Nextstrain) and Nextstrain Vars (all variants shown on Nextstrain).

Methods

Data was downloaded from Nextstrain.org, file ncov.json.
