Description

This composite of tracks represents GeneHancer1, a database of human regulatory elements (enhancers and promoters) and their inferred target genes, which is embedded in GeneCards, the human gene compendium2.

The GeneHancer view in the composite shows the regulatory elements.

We integrated >1 million regulatory elements from seven different genome-wide databases:

(A) The ENCODE project Z-Lab Enhancer-like regions (version v3)
(B) The Ensembl regulatory build (version 92)
(C) The FANTOM5 atlas of active enhancers
(D) The VISTA Enhancer Browser
(E) The dbSUPER super-enhancers
(F) The EPDnew promoters
(G) The UCNEbase ultra-conserved noncoding elements.

Employing an integration algorithm that aims to remove redundancy, GeneHancer shows ~250k integrated candidate regulatory elements ("GeneHancers"), each assigned an annotation-derived confidence score. The GeneHancers that are derived from more than one information source are defined as "elite" GeneHancers.

The Interaction view in the composite shows the associations between GeneHancers (in the GeneHancer view) and their target genes (in the Gene TSS view).

Gene-GeneHancer associations, and their likelihood-based scores, were generated using information that helps link regulatory elements to genes:

(A) eQTLs (expression quantitative trait loci) from GTEx (version v6p)
(B) Capture Hi-C promoter-enhancer long range interactions
(C) FANTOM5 eRNA-gene expression correlations
(D) Cross-tissue expression correlations between a transcription factor interacting with an enhancer and a candidate target gene
(E) Distance-based associations, including several approaches: (1) Nearest neighbors, where each GeneHancer is associated with its two proximal genes; (2) Overlaps with the gene territory (Intragenic); (3) Proximity to the gene TSS (<2kb).

The associations that are derived from more than one information source are defined as "elite" associations, which leads to the definition of the "double elite" dataset - elite gene associations of elite GeneHancers.

More details are provided at the GeneCards2 information page.

Methods

For a full description of the methods used, refer to the GeneHancer manuscript1.

Source data for the GeneHancer version 4.8 was downloaded during May2018. See Description above for source data versions.

Display conventions and Configuration

Each GeneHancer is identified by a GeneHancer id. For example: GH0XJ101383 is located on chromosome X, with starting position of 101,383 kb. Based on the id, one can obtain full GeneHancer information, as displayed in the Genomics section within the gene-centric web pages of GeneCards
