Description

This track shows genetic variants likely affecting gene expression in multiple human tissues. Specifically, the data displayed are the gene expression quantitative trait loci within 1MB of gene transcription start sites (cis-eQTL's) identified as causal in the Genotype-Tissue Expression (GTEx) project V6 release by CAVIAR analysis (95% credible set). The track shows eQTL's identified in 44 GTEx tissues, along with the effect size on gene expression (in FPKM), along with the probability the eQTL is in the 95% credible set.

NOTE: This track is under development, subject to change (and input is welcome!)

Display Conventions

Gene/variant pairs occuring in multiple tissues are merged into a single item in the display. The item label provides the tissue name or number of tissues. Mouseover displays effect size and tissues affected by the eQTL. The item color indicates the greatest effect size noted for the eQTL:
    fuchsiahigh positive
    redmoderate positive
    bluemoderate negative
    tealhigh negative
