Description

MANE, Matched Annotation from the NCBI and Ensembl-EBI, provides a matched set of well-supported transcripts for human protein-coding genes and defines one representative transcript for each gene. This data is intended to be a common reference transcript dataset, including one or a few key transcripts for each gene where RefSeq and Ensembl/GENCODE transcripts are identical in length and sequence, and completely match the human reference genome sequence. To pick representative transcripts, the MANE collaborators developed computational methods to evaluate transcript expression levels, protein conservation, support from archived transcript submissions, clinical relevance, and other factors. Complex genes are subject to review by annotation experts from both groups and often make improvements to both annotation sets. This minimal, identically annotated transcript set will eliminate the need to choose between RefSeq and Ensembl/GENCODE datasets for genomic analyses. This should make it easy for researchers who currently prefer one dataset over the other to exchange data or translate coordinates (or HGVS variation expressions) between RefSeq and Ensembl annotation results. Note that this data (from MANE 0.5) is a beta release and only covers 53% or protein coding genes. Both RefSeq and Ensembl will continue to provide a rich set of alternate transcripts in the RefSeq-All and the GENCODE tracks.

Data Access

The raw data can be explored interactively with the Table Browser, or the Data Integrator. For computational analysis, genome annotations are stored in a bigGenePred file that can be downloaded from the download server. Regional or genome-wide annotations can be converted from binary data to human readable text using our command line utility bigBedToBed which can be compiled from source code or downloaded as a precompiled binary for your system. Instructions for download can be found in the utilities directory. The utility can be used to obtain features within a given range, for example:

bigBedToBed -chrom=chr6 -start=0 -end=1000000 http://hgdownload.soe.ucsc.edu/gbdb/hg38/mane/mane.0.5.bb stdout

Please refer to our Data Access FAQ for more information or our mailing list for archived user questions.
