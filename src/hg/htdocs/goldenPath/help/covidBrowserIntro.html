Introduction to the SARS-CoV-2 Genome Browser

The UCSC Genome Browser is an open-source, interactive sequence visualization tool that has been a cornerstone of genomics for more than 20 years; cited in more than 37,000 scientific articles. It allows for cross referencing of research, clinical, and epidemiology data against the SARS-CoV-2 reference genome.
