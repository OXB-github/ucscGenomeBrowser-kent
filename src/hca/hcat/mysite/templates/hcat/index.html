Tuatara: presubmission data tracking

About

Suppose someone had a question about the data roadmap, "What is the current status of single nuclei-seq? I'd like to know when we can expect it so I can prioritize pipeline development, and from who so I can read papers from the labs."

Spreadsheets are typically used to track things, but they have some drawbacks. Using a relational database allows us to expand the tracking as necessary.

Tuatara is built with Django as an interface to this relationship database. An administrative layer over the database allows the brokering groups to maintain the relational tables.

Programmatic access to the database through a RESTful API is available to make Tuatara more useful to others.

API

A RESTful API allows one to query for current status of projects. Combined with the tracker.data.humancellatlas.org API, a full dashboard can be built to show all data known and in progress. Not all data is available through the API due to GDPR constraints, but enough data to make informed decisions based on known projects is available.
